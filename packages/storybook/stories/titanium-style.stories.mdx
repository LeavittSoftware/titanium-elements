# Titanium Styles

Individually export styles for targetd use

## How to use

```bash
npm i @leavittsoftware/titanium-styles
```

```js
import { h1, h2, h3, h4, h5, ellipsis, a, p, button } from '@leavittsoftware/titanium-styles';
```
