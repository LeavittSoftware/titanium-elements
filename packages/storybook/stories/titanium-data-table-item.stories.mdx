# Titanium Data Table Item

A data table element to organize row data and handle row selection.

row-item positioning attributes:

- right
- desktop
- large
- center
- width - ex. "140px"

## How to use

```bash
npm i @leavittsoftware/titanium-data-table
```

```js
import '@leavittsoftware/titanium-data-table/lib/titanium-data-table-item';
```
