# Leavitt Person or Company Select

Single select input that searches both Leavitt Group employees and companies

## How to use

```bash
npm i @leavittsoftware/leavitt-elements
```

```js
import '@leavittsoftware/leavitt-elements/lib/leavitt-person-company-select';
```
