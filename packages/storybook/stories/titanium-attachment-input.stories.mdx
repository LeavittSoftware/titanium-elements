# Titanium Attachment Input

Material design inspired file input.

## How to use

```bash
npm i @leavittsoftware/titanium-attachment-input
```

```js
import '@leavittsoftware/titanium-attachment-input';
```

## Demo

export const TitaniumAttachmentInputStory = ({ data, ...args }) =>
  html`
    <div style="margin-bottom:8px">
      <titanium-attachment-input label="Choose File" placeholder="Select a file" required validationMessage="File is Required"></titanium-attachment-input>
    </div>
    <div style="margin-bottom:8px">
      <titanium-attachment-input
        label="Choose File"
        placeholder="Select a file"
        .attachment=${{ Name: 'preselected', Extension: 'pdf' }}
        required
        validationMessage="File is Required"
      ></titanium-attachment-input>
    </div>
    <h2>Controlled</h2>
    <titanium-attachment-input
      .label=${args['label'] || 'Choose File'}
      .placeholder=${args['placeholder'] || 'Select a file'}
      .validationMessage=${args['validationMessage'] || 'File is Required'}
      .allowedFileType=${[]}
      ?required=${args['required']}
      ?disabled=${args['disabled']}
      @change=${args.change}
      @file-type-error=${args.fileTypeError}
    ></titanium-attachment-input>
  `;
