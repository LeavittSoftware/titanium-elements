# Titanium Snackbar

Material design inspired snackbar.

Only one snackbar is intended to be used per site, typically added to the site root page such as my-app.  Use TitaniumSnackbarSingleton esm export to fetch the instance of the snackbar when you need to use it.

## How to use

```bash
npm i @leavittsoftware/titanium-snackbar
```

```html
my-app.html

import '@leavittsoftware/titanium-snackbar';
...
...
<titanium-snackbar></titanium-snackbar>

```

To use anywhere in the application:
```js
import { TitaniumSnackbarSingleton as AppSnackbar } from '@leavittsoftware/titanium-snackbar';
AppSnackbar.open('Hello world');
```
