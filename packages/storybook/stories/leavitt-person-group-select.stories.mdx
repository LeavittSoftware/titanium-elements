# Leavitt Person or Group Select

Single select input that searches both Leavitt Group employees and groups

## How to use

```bash
npm i @leavittsoftware/leavitt-elements
```

```js
import '@leavittsoftware/leavitt-elements/lib/leavitt-person-group-select';
```
