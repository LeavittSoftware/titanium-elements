# Titanium Input Validator

A wrapper for elements you might want validated that do not come with built-in validation

## How to use

```bash
npm i @leavittsoftware/titanium-input-validator
```

```js
import '@leavittsoftware/titanium-input-validator';
```

## Demo

export const TitaniumInputValidatorStory = ({ inputText = '', ...args }) => html`
  <p>The input below should only return valid when the input is 'dog'</p>
  <titanium-input-validator .evaluator=${() => inputText === 'dog'} validationMessage="Input must say dog">
    <input
      @input=${event => {
        inputText = event.target.value;
      }}
    />
  </titanium-input-validator>
  <button
    @click=${() => {
      document.querySelector('titanium-input-validator').reportValidity();
    }}
  >
    Validate
  </button>
`;
